    Welcome to GridWatch
    
      
    
        A live emergency-dispatch simulator. Every metric on the right is
        a real measurement of an advanced data structure running under
        the hood.
      
    
      
      
        
    1. The map is a small grid city. H hospitals,
            F fire stations, P police.
      2. 
        
    2. Incidents spawn as pulsing dots — yellow / orange / red for low / medium / high severity.
      3. 
        
    3. The dispatcher uses a quadtree to find nearby idle units, then a
            Fibonacci-heap-backed Dijkstra to pick the fastest route.
      4. 
        
    4. Watch PQ ops, dispatch calls, Huffman ratio, and
            SA suffixes tick up in real time as the city handles traffic.
      5. 
        
    5. Type a street fragment in the search box to see the trie autocomplete; toggle
            fuzzy for the BK-tree spelling-tolerant search.
      6. 
      
    
      
    
        Press ? at any time to see this again. Full concept walkthrough is in
        docs/CONCEPTS.md.
